## Gebühren

Trades auf dem Hyperdash Terminal verursachen eine Builder-Gebühr von 1,5bp (0,015%) zusätzlich zur wettbewerbsfähigen Basis-Gebührenstruktur von Hyperliquid (https://hyperliquid.gitbook.io/hyperliquid-docs/trading/fees).

Trader sind berechtigt für Gebühren-Cashbacks basierend auf ihrer Stufe, die durch ihr Handelsvolumen bestimmt wird.

Handelsvolumen und Quests schalten weitere Stufen frei, die Ihnen einen größeren Anteil an Empfehlungsbelohnungen und Cashback ermöglichen. Empfehlungsraten gelten nur für direkte Empfehlungen.

### **Empfehlungen**

Empfehlungen werden auf mehreren Ebenen verdient.

- Stufe 1 (direkte Empfehlungen): Basierend auf Ihrer Stufe, z.B. Wood: 10%
- Stufe 2 (Empfehlungen von Empfohlenen): 4% von Stufe 1
- Stufe 3 (Empfehlungen von Empfohlenen von Empfohlenen): 1% von Stufe 2

Sie können Ihren Empfehlungscode und alle Belohnungen, die Sie verdienen, über den Belohnungsbereich auf der Profilseite (https://hyperdash.com/profile) beanspruchen sowie Ihren Empfehlungsgeber zuweisen. Sie können auch Ihr benutzerdefiniertes Profilbild von dieser Seite aus festlegen.
