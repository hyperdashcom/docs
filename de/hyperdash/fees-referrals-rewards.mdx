## Gebühren

Trades auf dem Hyperdash-Terminal verursachen eine Builder-Gebühr von 1bp (0,01%) zusätzlich zur wettbewerbsfähigen Basis-Gebührenstruktur von Hyperliquid (https://hyperliquid.gitbook.io/hyperliquid-docs/trading/fees).

Trader sind basierend auf ihrer Stufe, die durch ihr Handelsvolumen bestimmt wird, für Gebühren-Cashbacks berechtigt.

Handelsvolumen und Quests schalten weitere Stufen frei, die Ihnen einen größeren Anteil an Empfehlungsbelohnungen sowie Cashback ermöglichen. Empfehlungsraten gelten nur für direkte Empfehlungen.

### **Empfehlungen**

Empfehlungen werden auf mehreren Ebenen verdient.

- Ebene 1 (direkte Empfehlungen): Basierend auf Ihrer Stufe, z.B. Wood: 10%
- Ebene 2 (Empfehlungen von Empfehlungen): 12% von Ebene 1
- Ebene 3 (Empfehlungen von Empfehlungen von Empfehlungen): 4% von Ebene 2

Sie können Ihren Empfehlungscode und alle verdienten Belohnungen über den Belohnungsbereich der Profilseite (https://hyperdash.com/profile) beanspruchen sowie Ihren Empfehler zuweisen. Sie können auch Ihr benutzerdefiniertes Profilbild von dieser Seite aus festlegen.
