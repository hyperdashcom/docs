**The Explore page allows you to to view featured Hyperliquid traders, and apply filters to searches.**

A collections of notable wallets have been tagged for your convenience.

**Tracking**

All wallets found on hyperdash can be tracked, and organised using lists. Lists can be given custom names and assigned a coloured icon.

Users can set an 'alias' to each tracked wallet and add them to created lists for easy tagging, tracking and management.

Tracking wallets allows you to view a live feed of their trades in the trading terminal.

### Filtering of Global Traders

You can filter across all traders on Hyperliquid by:

- Trader Type
  - Algo
  - Scalp
  - Intraday
  - Swing
  - Position
- Account Value: total account value in perps account.
- PnL (all-time): Net Profit and Loss.
- Win Rate (%): The percentage of profitable trades taken.
- Sharpe Ratio : A measure of traders risk-adjusted returns.
